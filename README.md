# OLA Ride Data Analytics Project

📌 Project Overview

This project focuses on analyzing ride-booking data from Ola, one of India’s leading ride-hailing platforms.
The goal is to extract meaningful insights related to ride trends, customer behavior, cancellations, and revenue patterns using data analytics techniques.

The project demonstrates how raw ride data can be transformed into actionable insights to improve business operations and decision-making.

🎯 Objectives
Analyze ride booking patterns and peak demand periods
Understand customer and driver behavior
Identify cancellation trends and their reasons
Evaluate revenue and payment preferences
Generate insights to improve operational efficiency

🛠️ Tools & Technologies
SQL – Data extraction and querying
Python – Data cleaning and analysis
Power BI – Dashboard and reporting
CSV – Data source

📊 Dataset Description

The dataset contains ride-level information such as:

Booking ID
Customer & Driver details
Ride status (Success / Cancelled / Incomplete)
Pickup & Drop locations
Ride distance and fare
Payment methods
Ratings (Customer & Driver)

Such datasets typically include thousands of ride records and multiple features, enabling deep analysis of booking patterns, cancellations, and customer experience.

📈 Key Analysis Performed

🚀 Ride Trends Analysis
Ride volume over time (daily)
Peak hours and high-demand periods

❌ Cancellation Analysis
Total cancelled rides
Customer vs driver cancellations
Reasons behind cancellations

🚗 Vehicle Performance
Most used vehicle types
Average ride distance per vehicle

👥 Customer Insights
Top customers based on ride frequency
Customer behavior patterns

💰 Revenue Insights
Revenue distribution
Payment method analysis (Cash, UPI, Card, Wallet)
